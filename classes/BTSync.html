setFolderHosts()

setFolderHosts(string $secret, array $hosts) : object

Sets one or several predefined hosts for the specified sync folder. Existing list of hosts will be replaced.

Hosts should be added as values of the ‘host’ parameter and separated by commas. Returns current hosts if set successfully, error code otherwise.

Parameters

string $secret

(Required) Folder secret
array $hosts

(Required) List of hosts. Host should be represented as “[address]:[port]”
